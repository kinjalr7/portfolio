## 🚀 Local Development Setup

Follow these commands to clone, configure, and boot the portfolio application on your local machine:

### 1. Prerequisites
Ensure you have **Node.js (v18.x or above)** and **npm** installed.

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/kinjalr7/Portfolio-.git
cd Portfolio-

# Install the dependencies
npm install
```

### 3. Environment Variables Configuration
To use the contact form fully, retrieve a free access key from [Web3Forms](https://web3forms.com/) and create a `.env.local` file in the root of the project:

```env
NEXT_PUBLIC_WEB3FORMS_ACCESS_KEY=your_access_key_here
```

### 4. Running the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

### 5. Compiling a Production Build
```bash
npm run build
npm run start
```
